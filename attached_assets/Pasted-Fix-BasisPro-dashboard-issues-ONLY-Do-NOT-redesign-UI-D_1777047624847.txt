- Some links invalid → bad UX / errors

Fix in ModuleView.tsx:

1. Add helper:

const isValidExternalUrl = (url?: string) => {
  if (!url) return false;
  try {
    const parsed = new URL(url);
    return parsed.protocol === "https:" || parsed.protocol === "http:";
  } catch {
    return false;
  }
};

2. Update links rendering:

- Valid URL → normal <a>
